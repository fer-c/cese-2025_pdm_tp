Detailed Description

Estructura que representa la máquina de estados para el debounce.

Esta estructura contiene el estado actual de la máquina de estados, un delay para el debounce y una variable que indica si el botón fue presionado.

Note
Se debe inicializar la máquina de estados con la función debounceFSM_init.
